WHAT IS UI UX ?

Every interaction that we do, everything that we perceive and everything that we are bound to remember about any application or be it a website is the kind of a unique experience we had while using it. So to make that worth for every user the User interface and the User Experience comes into picture. Generally referred to as Ui / Ux all together. But now the question comes up is how we can find a difference between these two as people generally perceive them as two similar kind of things .

So let me help you clear that up and start knowing about them from the very start: User Interface - As the term itself says it is the interface that we are trying to design for our software. So, to put up in simple terms it is the visual designing done by a Ui designer by keeping the colours, typography, different layouts and many more things in mind. User Experience: It is how we design the interface for the user to interact with. For this the designer needs to perform a good amount of research about his targeted audience to know about what might be the most convenient interaction that every group of users can have.It also has the phase where in the designer creates some wireframes and prototypes so that the stakeholders do have a clear glimpse of the entire software and how it will turn out to be.

So overall every Ui Ux designer always prioritises the user at first. Let's look at a example, As we all browse social media applications it is quite familiar to all of us that we only reshare or repost some posts from different accounts if they are visually appealing. If there are 2 posts which have the same quote but one of them has a really amazing graphic design as its background and the other one is just some random image with a lot of pixel drop. Itâ€™s way too obvious that if we were to reshare one of the posts we would go with the 1st one as what's appealing to the eyes is what catches our attention. This clearly explains why we should actually focus upon having the best possible visuals for the user.
