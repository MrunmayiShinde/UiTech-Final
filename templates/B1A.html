DIFFERENCE BETWEEN UI UX

Any application designed for a particular task is performing really well in it but has a really simple interface it does not matter much there(eg-banking application). But, when you are trying to build a brand as a company or trying to grab user attention it is really necessary that we have some eye catchy visuals that might make the user want to spend more time on the application. Having a website with tons of user interactions but the users are unaware about the usage might be a real setback for the company. So the key of making every application user friendly is using the simplest and the most relevant interactions that the users are aware of or might have used before. This creates a sense of familiarity in the minds of users while they are using the application.
